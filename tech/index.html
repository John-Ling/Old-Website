Tech

Tech that I use

As a person basically glued to the computer, it makes sense that I would have used lots of different softwares and setups throughout the years. Some aspects of my current system I find are quite interesting so I thought it would be cool to share my setup.

Software

More details can be found at my Github

OS: I primarily use my computer for programming so using Linux over Windows means I'm not missing out on much. I daily drive Linux not out of a love of FOSS (I think FOSS is great don't get me wrong) but because I can customise my desktop to look cool. Yes I'm that vapid.

WM: Hyprland. I used to use KDE Plasma with Krohnkite for tiling. However nowadays you'll find me using Hyprland for its incredible animations.

Terminal: I use Alacritty. No reason in particular. I just like the name. I also use the ZSH shell with 'Oh my Zsh' and syntax highlighting with the Powerlevel10K theme.

Code Editor: I use VSCode. Again no reason in particular. It was just what I started with. For languages like C# or Java, I'll break out the Visual Studio 2022 and Intellij but for Javascript, Python and C/C++, VSCode is enough.

Hardware

I have two computers: my laptop and my PC. I run Windows 10 on my laptop and Linux on the PC.

Laptop

I use a Thinkpad T15 I bought in 2021. While it was not my first choice as I was really gunning for an AMD T14, it's been great for my schoolwork and programming. It has a quad-core I5, 16GB RAM and a 512GB NVME SSD. Nothing more to say really, it's a solid laptop.

PC

Here's where things get really interesting. Although I primarily use Linux on the PC, I use Windows 10 in a virtual machine using KVM for certain Linux-incompatible games and CAD applications like Fusion 360. I use GPU passthrough to achieve good graphics performance. I also run a MacOS VM just in case I decide to go into IOS development and because I can.

PC Specs:

  • Ryzen 5 2600
  • GTX 1070
  • RX 580
  • 16GB 3200MHz RAM
  • Asus ROG B450-F Gaming Mobo
  • NZXT Kraken X53 AIO
  • 2x 512GB NVME SSDs
  • 256GB and 128GB SATA SSD
  • Noctis 450 case
  • 4x BeQuiet 120mm fans
  • 2x 140mm white LED Aigo fans
