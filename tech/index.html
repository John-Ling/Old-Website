Tech that I use

Software

More details can be found at my Github

OS: I primarily use my computer for programming so using Linux over Windows means I'm not missing out on much. I daily drive Linux not out of a love of FOSS (I think FOSS is great don't get me wrong) but because I can customise my desktop to look cool. Yes I'm that vapid.

WM: Hyprland. I used to use KDE Plasma with Krohnkite for tiling. However nowadays you'll find me using Hyprland for its incredible animations.

Terminal: I use Alacritty. No reason in particular. I just like the name. I also use the ZSH shell with 'Oh my Zsh' and syntax highlighting with the Powerlevel10K theme. I also use the "Dracula Plus" colour scheme.

Code Editor: I use VSCode. Again no reason in particular. It was just what I started with. For languages like C# or Java, I'll break out the Visual Studio 2022 and Intellij but for Javascript, Python and C/C++, VSCode is enough.

Hardware

I have two computers: my laptop and my PC both running Linux.

Laptop

I use a Thinkpad T15 I bought in 2021. While it was not my first choice as I hoping to get an AMD T14 but it was sold out for months, it's been great for my schoolwork and programming. It has a quad-core 10th gen I5, 16GB RAM and a 512GB NVME SSD. Nothing more to say really, it's a solid laptop.

PC

I used to use a KVM + GPU passthrough setup with my PC to create high-performance Windows VMs for gaming. However right now I simply dual boot Windows 10 and Linux. I use Linux for most of my programming and general computer work while I use Windows for gaming, CAD and some Windows-exclusive programming.

Keyboard

While I don't really use fancy peripherals (I usually pick the cheapest IPS monitor and gaming mouse), I do have a custom keyboard I made back in 2021. Since I do a lot of typing I considered it a worth while investment. For the keyboard nerds reading this. I used Gateron Milky Reds with PBTfans Dolch keycaps. Taking a page from the adult film industry, I added lube to everything from the switches to the stabilisers. I also applied various mods to the board such as the Holee and tape mod. This keyboard is a dream to use and cost a little more than what you'd pay for a nice gaming keyboard at least in Malaysia.
